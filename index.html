Worth The Trip

Web application for fast and effective road-trip planning.

This application was built at TAMUHack 2014, Texas A&M's first annual hackathon, with Karrie Cheng, Laura Vaughan, and Chih-Yen Chang. WorthTheTrip utilizes the Google Maps and Google Positions APIs to provide recommendations for iconic stops along the way from a given start and end location using a specialized algorithm. I was responsible for the front-end (web) development. WorthTheTrip was built using Twitter Bootstrap.

Plaza

Local Networking Application for Android Devices

This application began development at Hackerbowl 2013 in College Station with Cody Taylor, Karrie Cheng, Cynthia Jyrkila, and Jerego Orlino. The idea behind Plaza is that social networking can and should be used to help people interact face to face, rather than hold them in isolation, especially in a collegiate setting. Users wishing to host an event, such as playing frisbee in the park, can anonymously post this event's information to the Board. All users within the area can see the Board, and are now welcome to join the host for a delightful evening in the park. I was in charge of designing the UI for this application.

This application does not yet have a demo.
